updateClassifier

  • updateClassifier(params: UpdateClassifierParams, callback?: Callback<Classifier>): ReadableStream | void
    • Defined in visual-recognition/v3-generated.ts:341

    Update a classifier.

    Update a custom classifier by adding new positive or negative classes (examples) or by adding new images to existing classes. You must supply at least one set of positive or negative examples. For details, see Updating custom classifiers. Important: You can't update a custom classifier with a free API key. If you have a free key, you can upgrade to a Standard plan. For details, see Changing your plan. Tip: Don't make retraining calls on a classifier until the status is ready. When you submit retraining requests in parallel, the last request overwrites the previous requests. The retrained property shows the last time the classifier retraining finished.

    Parameters

      The parameters to send to the service.
